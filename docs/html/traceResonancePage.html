Resonance absorption

This example is very similar to Trace through mesh and absorb energy via bremsstrahlung. As it is the case, only the part of the code that differ will be discussed here. Please make sure to read the Trace through mesh and absorb energy via bremsstrahlung first.

Key differences

First a raytracer::Marker is initialized to mark the cells where a ray got reflected. Theses are the only cells where the resonance absorption takes place:

Marker reflectedMarker;

This raytracer::Marker is then used in raytracer::SnellsLaw as this is the model that does decide whether a rey reflected or not:

SnellsLaw snellsLaw(
densityMeshFunction,
temperatureMeshFunction,
ionizationMeshFunction,
leastSquareGradient,
spitzerFrequency,
&reflectedMarker
);

The key difference is which model gets used in raytracer::AbsorptionController. In this case it is raytracer::Resonance:

AbsorptionController absorber;
Resonance resonance(leastSquareGradient, reflectedMarker);
absorber.addModel(&resonance);
